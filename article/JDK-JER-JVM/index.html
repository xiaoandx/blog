01-JDK，JRE，JVM有什么区别？

2021-05-02 Java面试 Java, 面试指南 0 评论

包含关系：JDK>JRE>JVM

JDK：

Java Development Kit，Java开发工具包，提供了Java的开发环境和运行环境。

包含了编译Java源文件的编译器Javac，还有调试和分析的工具。

JRE：

Java Runtime Environment，Java运行环境，包含Java虚拟机及一些基础类库

JVM：

Java Virtual Machine，Java虚拟机，提供执行字节码文件的能力

所以，如果只是运行Java程序，只需要安装JRE即可。

另外注意，JVM是实现Java跨平台的核心，但JVM本身并不是跨平台的，

不同的平台需要安装不同的JVM
